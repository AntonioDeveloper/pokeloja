# Pokeloja

Exercício de React + Typescript com o objetivo de fazer uma página 
de catálogo de pokemóns!

### Foram utilizados
- React
- Typescript
- Styled Components
- [PokeAPI](https://pokeapi.co/)

### Rode o projeto com:
- npm install e em seguida...
- npm run dev
